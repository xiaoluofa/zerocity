# 时间金钱

| 时间（单位分钟）  | 类型 | 奖励  |
| --------- | -- | --- |
| 30        | 天  | 100 |
| 60        | 天  | 125 |
| 120（2小时   | 天  | 150 |
| 480（6小时   | 周  | 175 |
| 960（16小时  | 周  | 200 |
| 1920（32小时 | 月  | 300 |
