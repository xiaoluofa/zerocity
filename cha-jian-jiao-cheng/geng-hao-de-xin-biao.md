# 更好的信标

计算公式如下：

```
     B       Beacon                    Blocks   x    TypeRange    x    LevelMultiplier  =   RangePerLevel
    NNN      3x3 Netherite               9      x       1.0       x          3.0        =          27
   DDDDD     5x5 Diamond                 25     x       0.8       x          2.0        =          40
  EEEEEEE    7x7 Emerald                 49     x       0.6       x          1.0        =          29.4
 GGGGGGGGG   9x9 Gold                    81     x       0.4       x          0.5        =          16.2
                                                                                                 ---------
                                                                                    Total Range =  112.6
```

TypeRange：

* 铁块: 0.3
* 金块: 0.3
* 绿宝石: 0.4
* 钻石: 0.6
* 下届合金块: 1.0 注：不要让我看到你最下层用下届合金块，很大怀疑你作弊了
