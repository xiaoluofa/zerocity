# 自定义前缀

**该功能仅赞助者可用，赞助教程**

**使用指令打开菜单**

```
/plt shop
```

点击如图位置，然后在文本框输入你想要的前缀即可

<figure><img src="https://s2.loli.net/2023/12/26/zQ52Lkf1lwFbWEK.png" alt=""><figcaption></figcaption></figure>

使用颜色代码，请访问[颜色生成器](https://mcg.tuanzi.ink/)然后直接复制到文本框粘贴即可，粘贴快捷键为CTRL+V
