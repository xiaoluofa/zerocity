# 付费传送

根据距离进行计费，目前每格价格为0.05，name为传送目标id

```
/goto preview name 预览价格
/goto new name 发送传送请求
/goto cancel 取消传送请求
/goto accept name 接受传送请求
/goto deny name 拒绝传送请求
/goto ignore name 屏蔽某人请求
/goto toggle 开启或关闭传送请求
```
